# learning-flasky-by-practice


## Setup

Add `.env` and `.flaskenv` files. For example,

```bash
# ./flaskenv
FLASK_APP=run.py
FLASK_ENV=development
FLASK_DEBUG=1
```

```bash
# .env
SECRET_KEY=419efc86-15ee-4ff2-865d-1937ca97bbdf
```

## Installation

```bash
$ pip install -U pip
$ pip install -r requirements.txt
```

## Usage

```bash
$ flask run
```
